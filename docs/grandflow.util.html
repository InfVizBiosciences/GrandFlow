class grandflow.util.timer.StageTimer[source]¶

Bases: object

Measure run times of different stages of the program

elapsed(stage)[source]¶

Return total time spent in a stage, which is the sum of the time spans between calls to start() and stop(). If the timer is currently running, its current invocation is not counted.

start(stage)[source]¶

Start measuring elapsed time for a stage

stop(stage)[source]¶

Stop measuring elapsed time for a stage.
